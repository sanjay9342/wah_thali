# Admin Guide

Use `/admin` for executive metrics, leakage alerts, and module navigation.

Core modules in this foundation:

- `/admin/orders` for order status visibility and next actions
- `/admin/customers` for Customer 360 style cards and WhatsApp actions
- `/admin/settings` for configurable business values

Production should add Supabase Auth guards and permission checks for Owner, Admin, Manager, Sales, Kitchen, Delivery, Finance, and Customer Support roles.
